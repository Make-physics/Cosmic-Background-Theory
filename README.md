概述
这项工作提出了一个基础理论，提出时空和物质不是原始实体，而是从一个更基本的前几何领域共同出现（称为“虚拟属性背景”）。这种出现的动力是由“背景赋予”机制控制的，由第一原理优化规则驱动。
该框架的核心优势在于其可证伪性：它为观测宇宙学提供了独特的、可测试的预测，包括特定的张量与标量比（r ≈ 0.004）和暗能量的动态状态方程（w(a) ≠ -1）。

核心论文
该理论建立在一些公理原则之上：
虚拟与实际的分离：背景潜力与表现现实之间的根本区别。
背景赋予机制：在“效率第一”原则指导下实现潜在状态的动态过程。
固有的宇宙“逻辑”（L）和“影响”（E）：宇宙被视为一个有机的、具有内在倾向的创造性过程。
根据这些原理，该理论自然地推导出关键的宇宙学参数（如标量光谱指数，n_s），并对即将到来的观测探测器（CMB-S4、欧几里得等）做出新颖的预测。
